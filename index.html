Différentes Informations Sur Les Plantes

Monstera Deliciosa

Origine : Forêts tropicales d'Amérique centrale.
Entretien :
- Lumière : Lumière indirecte vive, évitez le soleil direct
- Humidité : Aime une humidité élevée, vaporisez régulièrement les feuilles.

Aloe Vera

Origine : Régions arides d'Afrique.
Entretien :
- Lumière : Lumière directe ou indirecte vive.
- Particularité : Gel aux propriétés médicinales.

Ficus Lyrata

Origine : Afrique de l'Ouest.
Entretien :
- Lumière : Lumière indirecte vive, évitez les changements brusques d'emplacement.
- Humidité : Aimé une humidité moyenne, évitez les courants d'air.

Basilic

Origine : Régions tropicales et subtropicales.
Entretien :
- Lumière : Plein soleil ou lumière vive.
- Particularité : Utilisation culinaire populaire.

Lavande

Origine : Régions méditerranéennes.
Entretien :
- Lumière : Plein soleil
- Particularité : Parfum agréable et propriétés calmantes.

Menthe

Origine : Régions tempérées du monde entier.
Entretien :
- Lumière : Plein soleil ou lumière partielle.
- Particularité : Très envahissante si plantée en pleine terre.
