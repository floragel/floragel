
# 🌱 Floragel — Naturel Essentiel

**Floragel** est une mini-entreprise étudiante (développée avec JA Québec) qui propose une solution innovante et naturelle pour l'arrosage automatique de vos plantes. 

## 💡 Le Concept
Notre mission est de faciliter l’hydratation des végétaux avec une solution simple, efficace et respectueuse de l’environnement. Le gel Floragel permet de maintenir vos plantes hydratées en toute autonomie pendant une durée garantie de **2 semaines**, vous permettant de partir en voyage l'esprit tranquille.

## ✨ Caractéristiques principales
- **Innovation écologique** : Un produit 100 % satisfaisant et environnemental utilisant des matériaux de qualité.
- **Efficacité prolongée** : Jusqu'à 14 jours d'autonomie pour vos plantes.
- **Adaptabilité** : Un dosage précis selon la taille de vos plantes :
  - 🌿 **Grande plante** : 3 gels.
  - 🪴 **Moyenne plante** : 2 gels.
  - 🌱 **Petite plante** : 1 gel.
- **Support numérique** : Intégration d'un chatbot intelligent (Botpress) pour répondre aux questions des clients en temps réel.

## 🛠️ Stack Technique
Le site web de Floragel a été conçu pour être moderne, responsive et interactif :
- **Frontend** : HTML5, CSS3 (Bootstrap 5), JavaScript.
- **Animations** : Librairies `animate.min.css`, `Owl Carousel` et `Parallax` pour une expérience fluide.
- **Interactivité** :
  - **EmailJS** : Formulaire de contact fonctionnel pour recevoir les retours clients.
  - **Botpress** : Chatbot personnalisé pour l'assistance utilisateur.
  - **Supports multimédias** : Visualiseur de documents intégré pour la présentation orale du DNB.
